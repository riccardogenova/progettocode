CSS - Cascading Style Sheets

CSS, acronimo di Cascading Style Sheets, è un linguaggio di stile utilizzato per descrivere l'aspetto e la formattazione di un documento HTML. In altre parole, CSS è ciò che permette di rendere le pagine web visualmente accattivanti e ben strutturate.

Principali caratteristiche di CSS:

  1. Separazione dei contenuti e della presentazione: CSS separa il contenuto HTML dalla sua presentazione. Ciò significa che puoi definire lo stile e il layout di una pagina web in un file CSS separato, anziché all'interno del codice HTML stesso. Questo favorisce la manutenibilità e la flessibilità del codice.
  2. Selettore e dichiarazione: In CSS, ogni regola è composta da un selettore e una o più dichiarazioni. Il selettore indica quali elementi HTML verranno stilizzati, mentre le dichiarazioni specificano quali stili verranno applicati a quegli elementi. 
                    
                    h1 {
                        color: blue;
                        font-size: 24px;
                    }
    In questo esempio, h1 è il selettore e color: blue; e font-size: 24px; sono le dichiarazioni.
  3. Cascading: Il termine "cascading" indica che più regole CSS possono essere applicate allo stesso elemento, e il browser determinerà quale regola applicare in base alla specificità e all'ordine delle regole. Questo consente una maggiore flessibilità nello stile delle pagine web.
  4. Risparmio di codice: CSS permette di definire uno stile una volta e applicarlo a più elementi HTML. Ad esempio, è possibile creare classi e ID personalizzati per applicare lo stesso stile a più elementi o gruppi di elementi.
  5. Supporto per la responsività: CSS offre funzionalità per rendere le pagine web responsive, cioè adattive a diverse dimensioni di schermo e dispositivi. Questo è fondamentale per garantire una buona esperienza utente su dispositivi mobili, tablet e desktop.

In sintesi, CSS è uno strumento essenziale per la progettazione di pagine web moderne, consentendo di controllare l'aspetto e la presentazione dei contenuti HTML in modo efficace e flessibile.
