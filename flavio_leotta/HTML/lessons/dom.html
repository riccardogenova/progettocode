DOM, acronimo di Document Object Model, è una rappresentazione gerarchica strutturata dei documenti HTML (e di altri documenti XML) utilizzata dai browser web per rappresentare e manipolare il contenuto delle pagine web. In pratica, il DOM è una rappresentazione del documento HTML che può essere manipolata con linguaggi di scripting come JavaScript. Ogni elemento HTML nella pagina è rappresentato come un "nodo" nel DOM, e questi nodi possono essere modificati, aggiunti o rimossi dinamicamente utilizzando JavaScript. ` nel tuo documento HTML, questo verrà rappresentato come un nodo nel DOM. Con JavaScript, puoi selezionare questo nodo e cambiare il suo testo, aggiungere attributi o persino rimuoverlo completamente dalla pagina.
