Introduzione a JavaScript

Con JavaScript, ciò che programmiamo viene eseguito dal PC. La console di Google Chrome ti permette di modificare un sito perché contiene un ambiente Node.

Partiamo dalla dichiarazione delle variabili, utilizzando i famosi var, let, e const.

Ad esempio:

const example = "questa è una stringa"
console.log(example); // Questa istruzione stampa il contenuto della stringa

Abbiamo anche il tipo di dato booleano, che può essere vero o falso:

const bol = true; // o false

Esistono gli operatori di confronto:

const bol == const example;

E gli operatori di identità, che confrontano sia il valore che il tipo:

const num1 === const num2;

Per verificare se i valori sono diversi, possiamo usare != o !==, ma è consigliato utilizzare === e !==.

Abbiamo anche gli operatori di confronto maggiore o minore:

num1 > num2
num1 < num2
num1 <= num2
num1 >= num2

Approfondimenti su Array e Variabili in JavaScript

Array

Allo stesso modo delle stringhe, gli array hanno vari metodi utili per manipolare i dati al loro interno. Ad esempio, il metodo .push("elemento") viene utilizzato per aggiungere un nuovo elemento all'array:

const array = ["a", "b", "c"];
array.push("d"); // Aggiunge "d" alla fine dell'array
console.log(array); // Output: ["a", "b", "c", "d"]

Esistono molti altri metodi per manipolare gli array, come .pop(), .shift(), .unshift(), .splice() e molti altri. La ricerca su questi metodi ti permetterà di scoprire le varie possibilità offerte da JavaScript per lavorare con gli array.

Variabili e Assegnazione

Quando si lavora con variabili in JavaScript, è importante comprendere come avviene l'assegnazione dei valori, specialmente quando si tratta di tipi di dati complessi come gli array.

Quando assegniamo una variabile a un'altra, come nel seguente esempio:

const studente1 = "pino";
const studente2 = studente1;

Stiamo assegnando il valore della variabile studente1 alla variabile studente2. Tuttavia, se modificassimo il valore di studente1 in seguito, studente2 non verrebbe influenzato, poiché stiamo copiando solo il valore, non il riferimento all'oggetto.

Inoltre, quando si lavora con gli array, è importante capire che gli array sono trattati come puntatori in memoria. Ad esempio, se copiamo un array e poi aggiungiamo un elemento al nuovo array, stiamo modificando l'array originale, poiché entrambi i nuovi e vecchi array puntano allo stesso spazio di memoria.

Per creare una copia vera e propria di un array senza condividerne lo spazio di memoria, possiamo utilizzare l'operatore spread [...nomeArrayDaCopiare]:

const bello = ["copia"];
const bello1 = [...bello]; // Copia l'array bello in bello1

In questo modo, bello1 sarà una copia indipendente di bello, e le modifiche a uno non influenzeranno l'altro.
