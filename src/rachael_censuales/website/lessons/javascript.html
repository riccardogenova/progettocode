JavaScript

Javascript è un linguaggio di scripting Client-Side, utilizzato quindi per sviluppare funzionalità e interattività all'interno di pagine web eseguite sul client anziché sul server. Il linguaggio di programmazione Javascript segue le specifiche e le regole stabilite da ECMAScript: uno standard che fornisce le linee guida per la sintassi, le caratteristiche e il comportamento, queste ultime vengono seguite per garantire la coerenza e la compatibilità tra le diverse piattaforme. Javascript è in grado di cambiare il contenuto HTML di una pagina, interagendo direttamente con gli attributi degli elementi, lo stile, gli eventi e soprattutto comunicando con il Web Server.

Gestione degli Eventi

Per gestire gli eventi HTML in modo organizzato e riutilizzabile, è consigliabile definire le funzioni di gestione degli eventi in una pagina JavaScript separata. Questo approccio rende il codice più chiaro, manutenibile e scalabile. Tra gli eventi più comuni utilizzati in HTML e JavaScript ci sono:

  • onchange: si verifica quando un elemento HTML subisce un cambiamento.
  • onclick: si verifica quando l'utente fa clic su un elemento HTML.
  • onmouseover: si verifica quando il cursore del mouse entra in un elemento HTML.
  • onmouseout: si verifica quando il cursore del mouse esce dall'area di un elemento HTML.
  • onkeydown: si verifica quando l'utente preme un tasto sulla tastiera.
  • onload: si verifica quando la pagina web ha completato il caricamento.

Statements

JavaScript è un linguaggio Case Sensitive (distingue minuscole e maiuscole), e si può definire come un insieme di istruzioni (statements) eseguite per ottenere un certo risultato. Gli statements comprendono:

  • Keywords: 'var','let', 'const', 'if','else', che hanno significati speciali e non possono essere utilizzati come indicatori.
  • Espressioni: combinazioni (semplici o complesse) di valori, operatori e variabili valutate in un singolo valore. es. "3 + 4" è un'espressione aritmetica che restituisce il valore "7".
  • Valori: dati o informazioni memorizzati all'interno di un programma JavaScript, come numeri, stringhe, oggetti, funzioni, ecc.
  • Operatori: simboli speciali utilizzati per eseguire operazioni sui valori. (Operatori aritmetici per operazioni matematiche, operatori di confronto, operatori di assegnazione, ecc.)

Elementi di JavaScript

Per poter utilizzare efficacemente un programma JavaScript, è essenziale comprendere i concetti di base su cui si fonda il linguaggio. Questi concetti forniscono le fondamenta necessarie per lavorare con JavaScript in modo efficace e per capire come il codice viene eseguito all'interno di un ambiente di programmazione web.

  • Identificatori: nomi utilizzati per variabili e funzioni. Il primo carattere deve essere sempre una lettera, un underscore o un $.
  • Variabili: possono contenere valori di ogni tipo, vengono dichiarate con la keyword "var". Possono essere locali (dichiarata all'interno di una funzione e quindi valida solo all'interno della stessa) o globali.
  • Tipi di Dato Primitivi: Numeri, Boolean(true o false), stringhe, oggetti
  • Costrutti: strutture o elementi sintattici del linguaggio che consentono di eseguire determinate azioni o compiti nel codice. Tipi di costrutti:

    Istruzioni di controllo del flusso

    Consentono di controllare l'esecuzione del programma in base a condizioni specifiche (if, else, switch).

    • Operatori di confronto: `==` confronta due variabili (uguale a)
    • Operatori di identicità: `===` : confronta sia il contenuto che il tipo
    • Operatore di maggiore e minore tra variabili: `>` , `<` , `>=` , `<=`
    • Diverso: `!==` (identicità) - `!=` (solo il contenuto)
    Operatori logici:
    • Or Logico `| |`: controlla almeno una condizione, se almeno una è vera, l'espressione risulta vera, se sono entrambe false, l'espressione risulta falsa. Es. num1 < num2 | | num3 < num4
    • And Logico `&&`: verifica che entrambe le condizioni siano vere. Es: 5 === 2 && 6>7 FALSE ; 5>2 && 6<7 TRUE ;
    • Not logico: inverte un valore booleano. !nomevariabile ci da il contrario; !!nomevariabile contrario del contrario(condizione di esistenza);

      • La condizione di esistenza di una stringa vuota è falsa perché non c'è un valore dentro, anche 0 non esiste; >0 sempre true Con la condizione di esistenza si verifica che il contenuto di una variabile sia presente. true + true = 2 true + false = 1
