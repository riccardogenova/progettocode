  • Oggi useremo la console del browser;
  • L'ambiente console di Chrome usa lo strumento node.js;
  • Posso dichiarare una costante o una variabile const/var;
  • Con = si fa un'assegnazione;
  • Gli operatori usano == per confrontare due variabili SOLO SUL CONTENUTO;
  • Invece === confronta SIA IL CONTENUTO CHE IL TIPO;
  • Poi c'è != o !== che fanno il solito confronto MA SOLO SUL CONTENUTO;
  • Poi ci sono < > per fare altri confronti SUL CONTENUTO;
  • Una funzione/metodo si dichiara con function nomeDellaFunzione () {}
  • Prima però bisogna dichiarare la costante con const eventname = ''
  • Gli operatori logici usano il simbolo || e servono per confrontare UNA CONDIZIONE O L'ALTRA, NON ENTRAMBE;
  • Esempio: 5 < 2 || 6 < 7 (5 è minore di 2 OPPURE 6 è minore di 7);
  • Questo avviene quando ALMENO UNA dei due o più operatori è vero;
  • Poi c'è && da usare se ENTRAMBE SONO VERE;
  • Poi c'è il cosiddetto Nod Logico;
  • Esempio: const vero = true
  • Se poi si mette ! ti offre il contrario di quella variabile;
  • C'è anche !! per verificare che il contenuto tra '' sia presente;
  • Il numero 0 in JS è sempre falso;
  • Ci sono anche i valori null e undefined che sono equivalenti al false
  • NaN serve a segnalare operazioni che JS non comprende;
  • L'algoritmo è un insieme di istruzioni per risolvere un problema;
  • L'algoritmo è illustrabile tramite un diagramma di flusso con varie figure geometriche;
  • Il costrutto più semplice per un algoritmo è if else
  • "Se......allora.....altrimenti...."
  • Può anche capitare che if sia usato senza else
  • C'è anche switch per creare varie condizioni, similmente a if else
  • Altre condizioni: while che serve a svolgere un'operazione finchè non accade qualcosa ricominciando da capo, e do che svolge un'azione ALMENO UNA VOLTA;
  • Si può usare il . (punto) per vedere alcune cose dentro la stringa;
  • Array è una variabile (se l'informazione deve essere modificata)/constante (se l'informazione non la vogliamo modificare) che permette di salvare una lista di informazioni;
  • Gli array iniziano la loro posizione dallo 0, 1, 2, ecc.
  • Esempio: const array = ['a','b','c']
  • L'ultima posizione è accessibile vedendo la sua lunghezza con array [array.length - 1]
  • Ricorda che la LUNGHEZZA è il numero degli oggetti che parte da 1, la POSIZIONE parte da 0;
  • Altro argomento: un Array può contenere altri array al suo interno, e tale array interno occuperà anch'esso una posizione 0, 1, 2, ecc.
  • Gli Array hanno anche molti metodi al loro interno;
  • RICORDA: il metodo è una funzione, la proprietà è un valore;
  • Invece per copiare un array (e dunque crearne uno nuovo) si usa const primaVariabile = [...secondaVariabile]
  • Poi per assegnare il valore/contenuto di una variabile ad un'altra variabile si fa attraverso const studente1 = 'mario bianchi' per dichiarare la prima variabile, poi let studente2 = studente1 allora diventerà studente 2 = mario bianchi
  • Questo per il testo, invece per i numeri si fa: let num = 1 per dichiararla, poi let num2 = num1 per assegnare il valore, e quindi num2 = 1
  • Per modificare una variabile basta fare nomeVariabile.push('nuovo elemento')
  • Questo farà sì che la variabile abbia anche il valore 'nuovo elemento'
