  • Si può affrontare un problema distinguendo il campo tecnico (UI Design) da quello di business;
  • Per 'UI Designer' si intende il creatore dell'interfaccia utente dei siti web;
  • Di solito si usa il codice HTML e CSS, invece bisogna partire dallo sviluppare degli strumenti alternativi per risolvere un problema;
  • Bisogna domandarsi: cosa devo usare per risolvere questo problema?
  • Gli strumenti sino ad ora visti sono di due tipi: dati e azioni;
  • Le azioni sono le funzioni;
  • I dati sono gli elementi che rimangono salvati (post salvati, chat, ecc.);
  • Per esempio nel generare gruppi da 3 persone, si usa un array in quanto dato: const student = ["Raffaele", "Gianluca", "Marco", "Sofia", "Noemi", "Riccardo"]; const groupBy = 3; /Il numero con cui si vogliono raggruppare gli studenti/ const tempoStudentsGrouped = [ ["Raffaele", "Gianluca", "Marco"], ["Sofia", "Noemi", "Riccardo"], ] ;
