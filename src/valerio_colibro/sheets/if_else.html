If ed Else su Javascript

In JavaScript, if e else sono costrutti condizionali utilizzati per eseguire blocchi di codice in base al risultato di una condizione booleana. Questi costrutti permettono di eseguire determinate istruzioni solo se una condizione specificata è vera (true), altrimenti è possibile eseguire un altro blocco di istruzioni quando la condizione è falsa (false).

Ecco come funzionano:

  1. if: L'istruzione if viene utilizzata per eseguire un blocco di codice solo se la condizione specificata è vera. Se la condizione è falsa, il blocco di codice all'interno dell'if non viene eseguito.

    var x = 10;

    if (x > 5) {

    console.log("x è maggiore di 5");

    }

  2. else: L'istruzione else viene utilizzata insieme all'if per eseguire un blocco di codice alternativo se la condizione dell'if è falsa.

    var y = 3;

    if (y > 5) {

    console.log("y è maggiore di 5");

    } else {

    console.log("y non è maggiore di 5");

    }

  3. else if: È possibile utilizzare l'istruzione else if per gestire più condizioni alternative.

    var z = 7;

    if (z > 10) {

    console.log("z è maggiore di 10);

    } else if (z >5) {

    console.log("z è maggiore di 5 ma non di 10");

    } else {

    console.log("z non è maggiore di 5");

    }

In questo esempio, il codice esegue la prima istruzione console.log perché la condizione dell'if è vera (z è maggiore di 5 ma non di 10). Se la condizione dell'if fosse stata falsa, l'istruzione else if successiva sarebbe stata verificata e, se anche quella fosse stata falsa, il blocco else finale sarebbe stato eseguito.
