  1. Apertura del file desiderato
  2. Effettuare le modifiche al file
  3. Aggiunta delle modifiche alla staging area
  4. Esegui il commit delle modifiche

Lavorare su un file condiviso con Git:

  1. Clonare il repository
  2. Creare un branch
  3. Lavorare sul file
  4. Aggiungere le modifiche
