Window Object

L'oggetto Window rappresenta la finestra del browser che contiene un documento HTML. È il punto di ingresso principale per interagire con il browser e il documento.

Proprietà Principali

Alcune delle principali proprietà dell'oggetto Window includono:

  • window.document: Restituisce il documento caricato nella finestra.
  • window.location: Restituisce l'URL dell'attuale pagina web.
  • window.navigator: Restituisce informazioni sul browser e sul sistema operativo dell'utente.
  • window.innerWidth e window.innerHeight: Restituiscono le dimensioni interne della finestra del browser.
  • window.localStorage e window.sessionStorage: Consentono di memorizzare dati localmente sul browser.

Metodi Principali

Alcuni dei metodi principali dell'oggetto Window includono:

  • window.alert(): Mostra un messaggio di avviso all'utente.
  • window.prompt(): Mostra un prompt di input all'utente.
  • window.confirm(): Mostra un dialogo di conferma all'utente.
  • window.open(): Apre una nuova finestra del browser o un nuovo tab.
  • window.close(): Chiude la finestra del browser corrente.
