CSS

Questa categoria include una serie di comandi utilizzati per stilizzare e formattare contenuti web utilizzando il linguaggio di stile CSS (Cascading Style Sheets).

Selezionatori:

Tipo: elemento { }

ID: #id { }

Classe: .classe { }

Attributo: [attributo=valore] { }

Pseudo-classe: elemento:pseudo-classe { }

Pseudo-elemento: elemento::pseudo-elemento { }


Proprietà di posizionamento:

Posizione: position: valore;

Top, Bottom, Left, Right: top: valore;, bottom: valore;, left: valore;, right: valore;

Z-index: z-index: valore;

Float: float: valore;

Chiarezza: clear: valore;


Dimensioni:

Larghezza, Altezza: width: valore;, height: valore;

Margine: margin: valore;, margin-top: valore;, margin-bottom: valore;, margin-left: valore;, margin-right: valore;

Riempimento: padding: valore;, padding-top: valore;, padding-bottom: valore;, padding-left: valore;, padding-right: valore;

Immagini di sfondo: background-image: url('percorso');, background-size: valore;, background-repeat: valore;


Tipografia:

Carattere: font-family: 'nome-font', sans-serif;

Dimensione del carattere: font-size: valore;

Grassetto, Corsivo: font-weight: valore;, font-style: valore;

Allineamento del testo: text-align: valore;

Decorazione del testo: text-decoration: valore;

Spaziatura delle lettere: letter-spacing: valore;

Spaziatura delle parole: word-spacing: valore;

Trasformazione del testo: text-transform: valore;


Colori e sfondi:

Colore testo: color: valore;

Sfondo: background-color: valore;

Opacità: opacity: valore;

Box-shadow: box-shadow: valore;

Bordo: border: valore;, border-radius: valore;

Outline: outline: valore;


Effetti di transizione e animazione:

Transizione: transition: proprietà durata timing-func;

Animazione: animation: nome-animazione durata timing-func iterazioni direzione fill-mode;

Keyframes: @keyframes nome-animazione { }


Altre proprietà utili:

Visualizzazione: display: valore;

Overflow: overflow: valore;

Cursori: cursor: valore;

Zona di testo ridimensionabile: resize: valore;

Visibilità: visibility: valore;

Contorno del testo: text-stroke: valore;

Riposizionamento del cursore: caret-color: valore;


Media Query:

@media (condizione) { }


Pseudo-classi utenti:

:hover

:active

:focus

:visited

:nth-of-type(n)

:first-child

:last-child

Skew(valore): skew(valore)

Flex-wrap: flex-wrap: valore;

Flex: flex: valore;

Display: display: grid;

Grid-template-columns: grid-template-columns: valore;

Grid-template-rows: grid-template-rows: valore;

Grid-gap: grid-gap: valore;

Grid-column: grid-column: valore;

Grid-row: grid-row: valore;

Grid-area: grid-area: valore;

Justify-items: justify-items: valore;

Align-items: align-items: valore;

Justify-self: justify-self: valore;

Align-self: align-self: valore;
