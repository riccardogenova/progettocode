JAVASCRIPT

Comand

  1. querySelector e querySelectorAll: Per selezionare elementi HTML utilizzando selettori CSS.
    • var elemento = document.querySelector('.classe'); // Seleziona il primo elemento con la classe 'classe'
    • var elementi = document.querySelectorAll('p'); // Seleziona tutti gli elementi <p>
  2. addEventListener: Per aggiungere gestori di eventi a elementi HTML.
    • elemento.addEventListener('click', function() { // Azioni da eseguire quando l'elemento viene cliccato });
    • -click: Si verifica quando un elemento viene cliccato.
    • -mouseover: Si verifica quando il cursore del mouse si sposta sopra un elemento.
    • -mouseout: Si verifica quando il cursore del mouse esce da un elemento.
    • -keydown: Si verifica quando viene premuto un tasto sulla tastiera mentre l'elemento ha lo stato di focus.
    • -keyup: Si verifica quando un tasto precedentemente premuto sulla tastiera viene rilasciato.
    • -submit: Si verifica quando viene inviato un modulo.
    • -change: Si verifica quando il valore di un elemento di input, come un campo di testo o un menu a discesa, cambia.
    • -focus: Si verifica quando un elemento ottiene lo stato di focus.
    • -blur: Si verifica quando un elemento perde lo stato di focus.
  3. getElementById: Per selezionare un elemento HTML utilizzando il suo ID.
    • var elemento = document.getElementById('id');
  4. return: Utilizzato nelle funzioni per restituire un valore specifico quando la funzione viene chiamata
    • function somma(a, b) { return a + b; }
    • var risultato = somma(3, 4); // La funzione restituisce 7 console.log(risultato); // Output: 7
  5. length: Quante lettere è composta la parola prima
    • var stringa = "Ciao"; console.log(stringa.length); // Output: 4
  6. classList.add viene utilizzato per aggiungere una classe a un elemento HTML, mentre classList.remove serve per rimuovere una classe da un elemento HTML.
    • Esempio:
      function aggiungiClasse() {
      var paragrafo = document.getElementById("paragrafo");
      paragrafo.classList.add("highlight");
      function rimuoviClasse() {
      var paragrafo = document.getElementById("paragrafo");
      paragrafo.classList.remove("highlight");
      }
  7. parseInt è una funzione in JavaScript che converte una stringa in un numero intero.
    • Esempio:
      var numeroStringa = "123";
      var numeroIntero =parseInt(numeroStringa); // numeroIntero sarà 123
