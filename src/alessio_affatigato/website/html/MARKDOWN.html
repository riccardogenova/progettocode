MARKDOWN

Markdown è un linguaggio di markup con una sintassi del testo semplice progettata in modo che possa essere convertita in HTML e in molti altri formati usando un tool omonimo. Markdown è spesso usato per formattare file README, per scrivere messaggi in forum di discussioni e per creare testo formattato utilizzando un editor di testo semplice.

  • Utilizzando # prima di una parola verrà visualizzato come titolo nella schermata di prevew, e se si voglio creare sottotitoli basta aggiungere un cancelletto ulteriore.
  • Per visualizzare un testo in grassetto digitare due volte, prima e dopo, i simboli " * " e " _ " .
  • Invece per il testo in corsivo si usano gli stessi simboli " * " e " _ ", ma solo una volta prima e alla fine della sezione di testo interessata.
  • Per inserire link si inserisce tra [] il testo che vuoi e l'URl tra "<>"
  • Per creare elenchi usare - a inizio riga per tutti i punti dell'elenco, oppure se " 1. " per numerare gli elenchi
  • Per creare tabelle inserire " ' " all'inizio e alla fine
