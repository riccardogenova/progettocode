Lezione sull'oggetto document in JavaScript

Introduzione:

In JavaScript, l'oggetto document rappresenta il modello di oggetti del documento (DOM) caricato nella finestra del browser. È l'oggetto principale con cui puoi interagire per manipolare il contenuto HTML di una pagina web.

Accesso agli elementi HTML:

1. Selezione di elementi HTML:

  • document.getElementById(id): Restituisce un riferimento all'elemento HTML con l'ID specificato.
  • document.getElementsByClassName(className): Restituisce una collezione di elementi HTML con la classe specificata.
  • document.getElementsByTagName(tagName): Restituisce una collezione di elementi HTML con il tag specificato.
  • document.querySelector(selector): Restituisce il primo elemento HTML che corrisponde al selettore CSS specificato.
  • document.querySelectorAll(selector): Restituisce una NodeList di tutti gli elementi HTML che corrispondono al selettore CSS specificato. 
const myElement = document.getElementById('myElement');
const elementsByClass = document.getElementsByClassName('myClass');
const elementsByTag = document.getElementsByTagName('div');
const firstElement = document.querySelector('.myClass');
const allElements = document.querySelectorAll('.myClass');

2. Manipolazione degli elementi HTML:

  • element.innerHTML: Restituisce o imposta il contenuto HTML dell'elemento.
  • element.textContent: Restituisce o imposta il testo contenuto nell'elemento.
  • element.style.property: Restituisce o imposta il valore di stile CSS di un elemento.
  • element.setAttribute(attribute, value): Imposta un attributo su un elemento HTML. 
myElement.innerHTML = '
Questo è un paragrafo aggiunto tramite JavaScript
';
myElement.textContent = 'Nuovo testo';
myElement.style.color = 'red';
myElement.setAttribute('data-id', '123');

Creazione di nuovi elementi HTML:

const newDiv = document.createElement('div');
const newText = document.createTextNode('Questo è un nuovo testo');
newDiv.appendChild(newText);
document.body.appendChild(newDiv);

Manipolazione del documento:

console.log(document.title);
console.log(document.URL);
document.title = 'Nuovo titolo';

Manipolazione degli eventi:

document.addEventListener('click', function(event) {
    console.log('Hai cliccato su:', event.target);
});

Conclusioni:

L'oggetto document in JavaScript è essenziale per manipolare il contenuto HTML di una pagina web. Con le sue varie proprietà e metodi, puoi selezionare gli elementi HTML, manipolarli, creare nuovi elementi e gestire gli eventi, rendendo la tua pagina web interattiva e dinamica. La padronanza di document è fondamentale per lo sviluppo web con JavaScript.
