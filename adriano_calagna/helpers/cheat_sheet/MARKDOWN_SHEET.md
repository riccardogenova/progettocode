# Comandi Markdown

## Testo Formattato

- _Testo in corsivo_: `*testo*` o `_testo_`
- **Testo in grassetto**: `**testo**` o `__testo__`
- ~~Testo barrato~~: `~~testo~~`
- `Testo con codice`: `` `testo` ``
- [Link](https://www.example.com): `[testo del link](URL)`

## Liste

- Lista non ordinata:
  - Elemento 1
  - Elemento 2
  - Elemento 3
- Lista ordinata:
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
