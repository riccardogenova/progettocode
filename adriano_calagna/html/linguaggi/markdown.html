Cosa è il markdown

Markdown Markdown è un linguaggio di formattazione leggero utilizzato per scrivere testo semplice formattato. È comunemente usato per formattare testo in documenti, pagine web, file README nei repository di software, e altro ancora, ed è progettato per essere facile da scrivere e da leggere anche in forma non elaborata.
