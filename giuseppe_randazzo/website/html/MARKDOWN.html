Cos'è Markdown

Markdown è un linguaggio di markup leggero creato con l'obiettivo di essere il più leggibile possibile. Con linguaggio di markup leggero, si intende una sintassi semplificata usata per aggiungere formattazione a un testo, con l'intento di mantenere la leggibilità del testo puro e facilitare la conversione in altri formati come HTML, linguaggi progettati per essere facili da scrivere e da leggere, rendendoli ideali per la documentazione, la scrittura su web e la comunicazione rapida.

Il concetto di "leggero" si riferisce alla semplicità e alla minimizzazione del codice di markup necessario per ottenere risultati di formattazione usano convenzioni testuali semplici. Spesso usato per formattare file di testo semplice in modo da includere elementi come intestazioni, elenchi, link e altro, con l'intenzione che questi file siano facilmente convertibili in HTML o in altri formati.

Markdown è diventato estremamente popolare per la scrittura di documenti, note, pagine web, e persino post in forum o commenti online, grazie alla sua semplicità e leggibilità.
