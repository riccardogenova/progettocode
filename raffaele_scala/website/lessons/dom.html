Il Document Object Model (DOM)

Il DOM, acronimo di Document Object Model, è una rappresentazione gerarchica dei documenti HTML, XML o XHTML che costituiscono una pagina web.

Immagina il DOM come una struttura a albero che rappresenta tutti gli elementi di una pagina web: 

        <html>
        └── <head>
            ├── <meta charset="UTF-8">
            ├── <meta name="viewport" content="width=device-width, initial-scale=1.0">
            └── <title>Spiegazione del DOM</title>
        └── <body>
            ├── <h1>Il Document Object Model (DOM)</h1>
            ├── <p>Il DOM, acronimo di Document Object Model, è una rappresentazione gerarchica dei documenti...
            └── <p>Immagina il DOM come una struttura a albero che rappresenta tutti gli elementi di una pagina web:</p>

Ogni elemento è un "nodo" nel DOM, e questi nodi possono essere manipolati dinamicamente attraverso codice JavaScript.

Con il DOM, puoi:

  • Accedere agli elementi della pagina
  • Modificare il contenuto degli elementi
  • Aggiungere o rimuovere elementi
  • Gestire eventi
