Common CSS Commands

  • color: Imposta il colore del testo.
  • background-color: Imposta il colore dello sfondo.
  • font-size: Imposta la dimensione del carattere.
  • font-family: Imposta il tipo di carattere.
  • font-weight: Imposta lo spessore del carattere (normale, grassetto).
  • text-align: Allinea il testo (sinistra, destra, centro, giustificato).
  • text-decoration: Aggiunge decorazioni al testo (sottolineato, barrato).
  • padding: Aggiunge spazio interno agli elementi.
  • margin: Aggiunge spazio esterno agli elementi.
  • border: Aggiunge un bordo agli elementi.
  • width: Imposta la larghezza di un elemento.
  • height: Imposta l'altezza di un elemento.
  • display: Specifica il tipo di visualizzazione (blocco, inline, inline-block).
  • float: Allinea un elemento a sinistra o a destra.
  • position: Specifica il metodo di posizionamento (statico, relativo, assoluto, fisso).
  • top: Specifica la posizione superiore di un elemento posizionato.
  • right: Specifica la posizione destra di un elemento posizionato.
  • bottom: Specifica la posizione inferiore di un elemento posizionato.
  • left: Specifica la posizione sinistra di un elemento posizionato.
  • z-index: Imposta lo stack order (ordine di sovrapposizione) degli elementi posizionati.
  • line-height: Imposta l'altezza della riga.
  • letter-spacing: Imposta lo spazio tra i caratteri.
  • word-spacing: Imposta lo spazio tra le parole.
  • text-transform: Trasforma il testo in maiuscolo, minuscolo o altre forme.
  • text-shadow: Aggiunge un'ombra al testo.
  • box-shadow: Aggiunge un'ombra al contenuto del box.
  • border-radius: Arrotonda gli angoli dei bordi.
  • outline: Aggiunge una linea intorno a un elemento senza occupare spazio aggiuntivo.
  • opacity: Imposta la trasparenza di un elemento.
  • background-image: Imposta un'immagine di sfondo.
  • background-repeat: Specifica se e come ripetere un'immagine di sfondo.
  • background-position: Imposta la posizione di un'immagine di sfondo.
  • background-size: Imposta le dimensioni di un'immagine di sfondo.
  • background-attachment: Specifica se un'immagine di sfondo scorre con il contenuto o rimane fissa.
  • list-style-type: Imposta il tipo di elenco puntato.
  • list-style-image: Imposta un'immagine personalizzata come marcatore per gli elenchi puntati.
  • list-style-position: Specifica la posizione del marcatore degli elenchi puntati.
  • cursor: Imposta il tipo di cursore quando si passa sopra un elemento.
  • overflow: Specifica cosa fare quando il contenuto di un elemento supera le dimensioni specificate.
  • overflow-x: Specifica cosa fare quando il contenuto di un elemento supera le dimensioni lungo l'asse X.
  • overflow-y: Specifica cosa fare quando il contenuto di un elemento supera le dimensioni lungo l'asse Y.
  • text-overflow: Specifica come gestire il testo che è troppo lungo per adattarsi a un elemento.
  • white-space: Specifica come gestire lo spazio bianco all'interno di un elemento.
  • text-indent: Imposta l'indentazione della prima riga di un blocco di testo.
  • vertical-align: Imposta l'allineamento verticale di un elemento inline o di una cella di tabella.
  • position: Specifica il metodo di posizionamento (statico, relativo, assoluto, fisso).
  • top: Specifica la posizione superiore di un elemento posizionato.
  • right: Specifica la posizione destra di un elemento posizionato.
  • bottom: Specifica la posizione inferiore di un elemento posizionato.
  • left: Specifica la posizione sinistra di un elemento posizionato.
  • z-index: Imposta lo stack order (ordine di sovrapposizione) degli elementi posizionati.
  • clear: Specifica su quale lato degli elementi float non sono consentiti altri elementi float.
  • float: Allinea un elemento a sinistra o a destra.
  • max-width: Imposta la larghezza massima di un elemento.
  • min-width: Imposta la larghezza minima di un elemento.
  • max-height: Imposta l'altezza massima di un elemento.
  • min-height: Imposta l'altezza minima di un elemento.
  • opacity: Imposta l'opacità di un elemento.
  • overflow: Specifica il comportamento dell'overflow del contenuto.
  • text-transform: Trasforma il testo in maiuscolo o minuscolo.
  • line-height: Imposta l'altezza della riga.
  • vertical-align: Allinea verticalmente un elemento inline o tabellare.
  • border-radius: Aggiunge angoli arrotondati a un elemento.
  • box-shadow: Aggiunge un'ombra a un elemento.
  • text-shadow: Aggiunge un'ombra al testo di un elemento.
  • box-sizing: Specifica il modello di box-sizing per un elemento.
  • cursor: Specifica il tipo di cursore al passaggio del mouse su un elemento.
  • background-image: Imposta un'immagine di sfondo per un elemento.
  • background-position: Imposta la posizione iniziale dell'immagine di sfondo.
  • background-repeat: Specifica come ripetere un'immagine di sfondo.
  • background-size: Imposta le dimensioni dell'immagine di sfondo.
  • background-attachment: Specifica se un'immagine di sfondo deve scorrere con la pagina o rimanere fissa.
  • transition: Specifica le transizioni per un elemento.
  • transform: Applica trasformazioni 2D o 3D a un elemento.
  • animation: Definisce una sequenza di animazioni.
  • keyframes: Definisce le regole per l'animazione.
  • backface-visibility: Specifica se il retro di un elemento deve essere visibile quando viene ruotato.
  • perspective: Imposta il punto di vista per le trasformazioni 3D.
  • user-select: Specifica se un utente può selezionare il contenuto di un elemento.
  • outline: Imposta una riga intorno a un elemento esterno, ma non aumenta le dimensioni dell'elemento.
  • outline-offset: Imposta lo spazio tra l'outline e il bordo dell'elemento.
  • word-wrap: Specifica se il browser può spezzare le parole per mantenere il layout.
  • white-space: Specifica come gestire lo spazio vuoto all'interno di un elemento.
  • text-overflow: Specifica cosa fare quando il testo di un elemento è troppo lungo per adattarsi all'area.
  • tab-size: Imposta la larghezza della tabulazione.
  • resize: Specifica se un utente può ridimensionare un'area di testo.
  • appearance: Permette di definire l'aspetto di un elemento (come i controlli di input) usando i valori nativi del sistema operativo.
  • filter: Applica effetti visivi (come sfocatura o regolazione del colore) a un elemento.
  • flex-direction: Specifica la direzione principale di un contenitore Flex.
  • justify-content: Allinea gli elementi lungo l'asse principale di un contenitore Flex.
  • align-items: Allinea gli elementi lungo l'asse trasversale di un contenitore Flex.
  • flex-grow: Specifica quanto un elemento flessibile dovrebbe crescere rispetto agli altri elementi flessibili all'interno del contenitore Flex.
  • flex-shrink: Specifica quanto un elemento flessibile dovrebbe ridursi rispetto agli altri elementi flessibili all'interno del contenitore Flex.
  • flex-basis: Specifica la dimensione di base di un elemento flessibile all'interno del contenitore Flex.
  • flex: Shorthand per flex-grow, flex-shrink, e flex-basis.
  • grid-template-columns: Specifica la larghezza delle colonne in un layout a griglia.
  • grid-template-rows: Specifica l'altezza delle righe in un layout a griglia.
  • grid-column-gap: Specifica lo spazio tra le colonne in un layout a griglia.
  • grid-row-gap: Specifica lo spazio tra le righe in un layout a griglia.
  • grid-gap: Shorthand per grid-row-gap e grid-column-gap.
