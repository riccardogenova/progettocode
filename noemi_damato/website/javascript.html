JAVASCRIPT

un linguaggio di programmazione usato principalmente per rendere le pagine web interattive. Puo' manipolare il contenuto HTML e CSS,

Variabili: Usate per memorizzare dati temporanei.

Funzioni: Blocchi di codice riutilizzabili che eseguono una specifica azione.

Condizioni: Strutture di controllo come if, else if, else per eseguire azioni in base a condizioni.

Cicli: Strutture di controllo come for, while per eseguire ripetutamente blocchi di codice.

Oggetti: Raccolte di dati e funzioni correlate.

Eventi: Interazioni utente come clic, hover, e inserimento di testo che attivano azioni nel codice.

Manipolazione del DOM: Modifica del contenuto, della struttura e dello stile degli elementi HTML e CSS sulla pagina.

Chiamate AJAX: Comunicazione asincrona con il server per ottenere o inviare dati senza ricaricare la pagina.

window e document sono oggetti fondamentali che forniscono accesso a funzionalita' cruciali nell'ambiente del browser :

window:

Rappresenta la finestra del browser. E' l'oggetto globale in JavaScript e contiene molte proprieta' e metodi utili.

  • window.alert(): mostra un messaggio di avviso,
  • window.location:fornisce informazioni sull'URL della pagina.
  • tolowindow.setTimeout():imposta un timer che esegue una funzione dopo un certo intervallo di tempo.

document:

Rappresenta il documento HTML corrente caricato nel browser. Fornisce metodi per accedere agli elementi HTML e manipolarli dinamicamente.

  • document.getElementById(): recupera un elemento HTML tramite il suo ID.
  • document.querySelector(): seleziona il primo elemento corrispondente a un selettore CSS.
  • document.createElement(): crea un nuovo elemento HTML.

addEventListener:

E' un metodo utilizzato in JavaScript per associare una funzione, nota come "gestore degli eventi", e' una tecnica fondamentale per rendere le pagine web interattive e per gestire le interazioni utente.

La sintassi di base e' la seguente:

  • element:

    L'elemento HTML a cui si desidera associare l'evento.

  • event:

    l nome dell'evento su cui si desidera che la funzione venga eseguita

  • function

    La funzione da eseguire quando si verifica l'evento.

Gli operatori in JavaScript possono essere divisi in tre categorie principali: confronto, logici e condizionali.

Operatori di confronto:

Gli operatori di confronto in JavaScript sono utilizzati per confrontare due valori e restituire un valore booleano (true o false) che indica se la relazione tra i valori e' vera o falsa. Questi operatori di confronto sono ampiamente utilizzati per creare condizioni e istruzioni di controllo del flusso nei programmi JavaScript.

Alcuni degli operatori di confronto piu' comuni:

  • Uguaglianza (==): Verifica se due valori sono uguali.
  • Uguaglianza stretta (===):Simile all'operatore di uguaglianza, ma verifica anche se i tipi dei valori sono gli stessi.
  • Non uguale (!=):Verifica se due valori non sono uguali. Questo operatore esegue una conversione di tipo debole prima del confronto.
  • Maggiore di (>), Minore di (`<`), Maggiore o uguale a (`>=`), Minore o uguale a (`<=`): Questi operatori confrontano due valori numerici per determinare se il primo valore è maggiore, minore, maggiore o uguale, o minore o uguale al secondo valore.

    • Operatori logici: Gli operatori logici sono utilizzati per eseguire operazioni logiche su valori booleani e restituire un risultato booleano.Gli operatori logici sono ampiamente utilizzati per creare espressioni condizionali e controllare il flusso del programma in base a determinate condizioni.

    Ecco gli operatori logici principali:

  • AND logico (&&): Restituisce true se entrambi gli operandi sono true, altrimenti restituisce false.
  • OR logico (||): Restituisce true se almeno uno degli operandi è true, altrimenti restituisce false.
  • NOT logico (!):Restituisce il valore opposto dell'operando booleano. Se l'operando è true, restituisce false; se l'operando è false, restituisce true.

    • Operatori condizionali: Gli operatori condizionali sono utilizzati per scrivere codice più conciso e chiaro quando si devono gestire condizioni semplici e prendere decisioni basate su valori booleani o di verità.

  • Operatore ternario (condizione ? espressione1 : espressione2):è un'abbreviazione di un'istruzione if-else. Se la condizione è vera, restituisce espressione1; altrimenti, restituisce espressione2.
      l'istruzione if viene utilizzata per eseguire un blocco di codice se una determinata condizione è vera. L'istruzione else permette di eseguire un blocco di codice diverso se la condizione dell'if è falsa.

  • Nullish coalescing operator (??): Restituisce il valore a destra se il valore a sinistra è null o undefined; altrimenti, restituisce il valore a sinistra.
      In JavaScript, null, undefined, e NaN sono tutti valori speciali utilizzati per rappresentare diversi tipi di situazioni o stati.
      • null:
      valore primitivo che rappresenta l'assenza di valore, utilizzato per indicare l'assenza di un oggetto o per azzerare una variabile di riferimento.
      • undefined:
      indica che una variabile è stata dichiarata ma non inizializzata o non ha un valore assegnato. utilizzato di solito come valore di default assegnato automaticamente a una variabile non inizializzata.
      • NaN (Not a Number):
      valore speciale, viene restituito quando si tenta di eseguire operazioni matematiche non valide,.

    addEventListener

    in JavaScript è un metodo utilizzato per associare un gestore degli eventi a un elemento HTML. Questo metodo consente di definire una funzione che verrà eseguita quando si verifica un determinato evento su quell'elemento. es: const myButton = document.getElementById('myButton'); myButton.addEventListener('click', function() { console.log('Il pulsante è stato cliccato!'); }); ---------------------------------------------------------------

    array

    In JavaScript, è una struttura dati che consente di memorizzare più valori in una singola variabile. Gli array possono contenere elementi di qualsiasi tipo di dato, inclusi numeri, stringhe, oggetti e anche altri array. Gli array in JavaScript supportano metodi mutabili e immutabili per manipolare i dati.

  • Metodi mutabili: modificano direttamente l'array originale.
    1. push(): Aggiunge uno o più elementi alla fine dell'array e restituisce la nuova lunghezza dell'array. Modifica direttamente l'array originale.
    2. pop(): Rimuove l'ultimo elemento dall'array e lo restituisce. Modifica direttamente l'array originale.
    3. splice(): Aggiunge o rimuove elementi dall'array modificando direttamente l'array originale.
  • Metodi immutabili: consentono di manipolare i dati senza modificare direttamente l'array originale.

    1. map():
      2. Questo metodo non modifica l'array originale, ma restituisce un nuovo array con i risultati delle chiamate alla funzione applicata a ciascun elemento dell'array. Restituisce un array `return`

        Sintassi:
      • elemento: L'elemento corrente che sta per essere trasformato.
      • index: L'indice dell'elemento corrente nell'array.
      • array: L'array a cui appartiene l'elemento corrente
    2. filter():Questo metodo prende un array e restituisce un nuovo array contenente solo gli elementi che soddisfano una determinata condizione definita da una funzione di filtro.
        Sintassi:
      • callback: È una funzione che viene chiamata per ogni elemento dell'array. Deve restituire true per mantenere l'elemento nell'array risultante o false per eliminarlo.
      • element: L'elemento corrente dell'array che viene processato dalla funzione di callback.
      • index (opzionale): L'indice dell'elemento corrente all'interno dell'array.
      • array (opzionale): L'array su cui è stato chiamato il metodo filter().
    3. reduce(): È un metodo che riduce un array di valori ad un unico valore, applicando una funzione di riduzione che accumula i risultati iterando su ogni elemento dell'array.
        4. Sintassi:
      • callback: È la funzione di riduzione che viene chiamata per ogni elemento dell'array. Riceve quattro argomenti:
      • accumulator: Il valore accumulato dopo l'esecuzione di tutte le chiamate precedenti alla funzione di callback o il valore iniziale, se fornito.
      • currentValue: L'elemento corrente dell'array che viene processato dalla funzione di callback.
      • index (opzionale): L'indice dell'elemento corrente all'interno dell'array.
      • array (opzionale): L'array su cui è stato chiamato il metodo reduce().
      • initialValue (opzionale): Un valore iniziale per l'accumulatore. Se non viene fornito, il valore iniziale sarà il primo elemento dell'array e la riduzione inizierà dall'elemento successivo.
    4. find():Trova il primo elemento in un array che soddisfa una determinata condizione definita da una funzione di ricerca. Restituisce il primo elemento trovato oppure undefined se nessun elemento soddisfa la condizione.
        5. Sintassi:
      • callback: L'elemento corrente dell'array che viene processato dalla funzione di ricerca.
      • index (opzionale): L'indice dell'elemento corrente all'interno dell'array.
      • array (opzionale): L'array su cui è stato chiamato il metodo find().
      • thisArg (opzionale): Un valore da passare come this quando viene eseguita la funzione di ricerca.

    classi

    in JavaScript sono come modelli per creare oggetti con proprietà e comportamenti simili. Usi la parola chiave `class` per dichiarare una classe, e all'interno puoi definire un costruttore `constructor` per inizializzare le proprietà dell'oggetto e metodi per definire il suo comportamento. Le istanze di una classe sono create usando la parola chiave `new`.
      Costruttore: Il costruttore è utile per garantire che ogni istanza della classe sia inizializzata correttamente quando viene creata.

      ex: class Persona { constructor(nome, età) { this.nome = nome; this.età = età; } }

      Nell'esempio sopra, constructor è il costruttore della classe Persona. Prende due parametri (`nome` e `età`), quando si crea un'istanza della classe il costruttore viene automaticamente chiamato con i parametri forniti e le proprietà dell'oggetto vengono inizializzate di conseguenza

    ------------------------------------------------------------------- Metodi: sono funzioni associate a oggetti. Possono essere definiti all'interno di una classe o direttamente su un oggetto. Possono accettare parametri e possono accedere alle proprietà dell'oggetto usando `this`. Sono utilizzati per definire il comportamento degli oggetti e per eseguire operazioni su di essi.
