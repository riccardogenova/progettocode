Cosa è Switch su Javascript?

In JavaScript, lo statement switch viene utilizzato per eseguire una serie di istruzioni differenti in base al valore di una variabile o di un'espressione. È una forma di controllo del flusso di esecuzione che permette di gestire multiple alternative in modo più compatto rispetto a una serie di istruzioni if...else.

Ecco un esempio:

switch (espressione) {

case valore1:

// istruzioni da eseguire se espressione è uguale a valore 1

break;

case valore2:

// istruzioni da eseguire se espressione è uguale a valore2

break;

// Altri casi possibili

default:

// istruzioni da eseguire se nessun caso precedente è stato soddifsfatto

}

Quando viene eseguito uno statement switch, JavaScript valuta l'espressione e confronta il risultato con i valori associati ai vari casi (case). Se trova una corrispondenza, esegue le istruzioni all'interno del blocco di codice associato a quel caso e interrompe l'esecuzione con l'istruzione break. Se nessun caso corrisponde al valore dell'espressione, viene eseguito il blocco di codice associato a default, se presente.

Ecco un altro esempio:

var giorno = 3;

var nomeGiorno;


switch (giorno) {

case 1:

nomeGiorno = 'Lunedì;

break;

case 2:

nomeGiorno = 'Martedi';

break;

case 3:

nomeGiorno = 'Mercoledì';

break;

// Altri casi possibili

default:

nomeGiorno = 'Giorno non valido';

}

console.log(nomeGiorno);

// Output: Mercoledi

In questo esempio, la variabile giorno ha il valore 3, quindi viene associato il valore 'Mercoledì' alla variabile nomeGiorno tramite lo statement switch.
